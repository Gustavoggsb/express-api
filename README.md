# Express API

This is a conceptual API, developed to serve as a basis for new projects.



  

### Tech

* node.js
* Express
* Bcrypt
* Body-Parser
* fs
* Jsonwebtoken
* Mongoose
* Nodemailer
* Path


### Installation

Install the dependencies and start the server.

```sh
$ npm install 
$ npm start
```
